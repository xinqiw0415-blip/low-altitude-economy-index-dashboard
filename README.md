# 低空经济事件与市场数据项目


## 数据分层

- `data/raw/`：采集到的原始响应，不人工修改。
- `data/interim/`：清洗、去重、标准化后的中间表。
- `data/processed/`：建模使用的数据集，后续生成。
- `config/`：股票池、政策来源等人工审核配置。
- `reports/`：质量审计报告。

## 一键运行

```powershell
powershell -ExecutionPolicy Bypass -File .\run_pipeline.ps1
```

## 启动可视化系统

```powershell
powershell -ExecutionPolicy Bypass -File .\run_dashboard.ps1
```

市场数据脚本默认读取 `config/stock_universe.csv`，将每个证券的原始 JSON 保存在 `data/raw/market/`，并输出统一长表 `data/interim/market_daily.csv`。重复运行时会覆盖同名原始响应并重新生成中间表，确保过程可复现。

> `stock_universe.csv` 是研究初始池，不代表最终样本。每个标的必须经过主营业务、低空经济相关事件和样本期可交易性审核后，才能进入正式指数。

## 当前限制

- 40条公司事件完成双人标注及裁决，其余公司事件与全部政策子事件仍依赖DeepSeek证据过滤。
- 新闻历史仅覆盖2026年4月至6月，未进入长期时序模型。
- 腾讯历史接口不提供成交额和换手率，正式论文前仍建议使用第二行情源交叉核验。
- 全样本动态因子指数用于历史解释；预测使用按月扩展窗口实时指数，二者用途不同。
- 预测AUC接近0.5，不能宣称具备稳定交易能力；统计关系也不能直接解释为因果关系。
