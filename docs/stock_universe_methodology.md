# 低空经济股票池方法 v1.0

## 研究边界

股票池依据国家发展改革委《低空经济及其核心产业统计分类（试行）》所界定的产业链思路划分为四类：低空航空器制造、低空运营服务、低空基础设施与信息服务、低空配套产品与服务。

## 纳入规则

同时满足以下条件才进入正式研究池：

1. A股普通股，样本期内存在可交易日行情；
2. 主营业务或公开披露项目可映射至上述四类之一；
3. 相关性可由定期报告、公司公告或政府项目文件验证；
4. 不是仅凭互动平台概念表述或短期市场标签纳入；
5. 保留证据URL、证据日期和审核状态。

## 分层

- `core`：直接从事航空器制造、无人机制造、通航运营、空管系统等核心业务；用于核心指数。
- `extended`：基础设施、规划咨询、零部件等间接业务；用于扩展指数和稳健性检验。
- `watch`：相关性尚待正式公告验证；不进入当前指数。

股票池采用版本化管理。历史回测必须使用当时已经上市且可交易的成分股，不能用未来上市公司回填历史。
