# 低空经济动态情绪指数与风险预警系统：阶段性总报告

## 项目定位

本项目从传统的“文本情绪分类 + 股价涨跌预测”升级为“LLM事件抽取 + 动态情绪指数 + 事件冲击检验 + 极端表现与波动风险预警”的研究与可视化系统。

系统不以普通日频涨跌预测为核心卖点，而是围绕低空经济政策和公司事件，识别市场状态、事件冲击、极端相对表现和波动风险。

## 数据基础

- 股票池：60只低空经济核心及扩展产业链A股。
- 市场数据：98,761条个股日频记录，2019-01-02至2026-06-15。
- 公司公告：41,432条公告元数据，2,226份有效公告文本，115条LLM公司事件抽取任务。
- 政策文本：18份官方政策源，136条政策子事件。
- 人工标注：40条公司事件样本完成双人标注及裁决。
- 最终事件库：46条接纳公司事件，123条接纳政策事件。

## 方法路线

1. 使用DeepSeek/LLM将公告和政策文本抽取为结构化事件。
2. 基于事件方向、强度、不确定性、新颖度和证据质量构造事件特征。
3. 构建政策支持、企业景气、置信度余额和有方向事件广度四个日频通道。
4. 使用动态因子模型构建低空经济动态情绪指数。
5. 使用分位数回归、VAR、局部投影和事件研究检验指数与市场关系。
6. 使用无前视实时指数和个股面板模型进行辅助预测。
7. 使用Streamlit构建可视化系统。

## 核心发现

- 动态因子模型成功收敛，PCA第一主成分解释率为81.39%。
- 动态指数与PCA基准相关系数为0.987，与等权指数相关系数为0.972。
- 分位数回归显示非对称关系：低、中分位收益系数为负，90%分位收益系数为正，FDR校正后仍显著。
- VAR中情绪冲击对超额收益的Granger检验p值为0.0046，反向检验不显著。
- 普通5日跑赢目标AUC约0.52，不能作为项目亮点。
- 极端相对强弱预测全样本AUC为0.537，事件窗口AUC为0.655。
- 波动率跳升预测全样本AUC为0.679，事件窗口AUC最高为0.697。
- 高置信度波动风险方向判断准确率最高约94.5%，但覆盖率为5%，且主要体现为风险规避型识别。
- 每日Top 5%波动风险命中率为43.5%，相对24.0%的基准发生率提升约81.3%。

## 可视化系统

系统包括五个页面：

- 研究总览：展示动态情绪指数、实时指数和市场指数。
- 指数拆解：展示政策支持、企业景气、置信度余额和事件广度。
- 事件时间线：查询公司公告事件和政策事件。
- 统计检验：展示分位数回归、局部投影、VAR和安慰剂检验。
- 预测解释：展示面板模型、Top关注名单、高置信度信号和SHAP解释。

启动命令：

```powershell
powershell -ExecutionPolicy Bypass -File .\run_dashboard.ps1
```

## 创新点

- 从“文本情绪打分”升级为“事件结构化抽取”，增强解释性和可追溯性。
- 将LLM抽取与统计模型分工：LLM负责结构化变量，统计模型负责指数构建和关系检验。
- 构造低空经济动态情绪指数，而非仅预测单日涨跌。
- 同时提供全样本解释指数和无前视实时指数，区分历史解释与预测评估。
- 将预测目标从普通涨跌改造为极端相对表现和波动风险预警，更符合事件冲击的真实作用机制。
- 用可视化系统整合事件证据、指数走势、统计检验和预测解释。

## 方法边界

- 公司事件人工标注样本仍较少，事件窗口结果需要更多事件验证。
- 普通涨跌预测能力有限，不能宣传为高准确率交易系统。
- 事件特征相对技术基线的稳定增量尚未完全证明。
- 高置信度准确率必须与覆盖率同时报告。
- 扩展产业链样本相关度不完全一致，论文中需区分核心样本和扩展样本。
- 所有结果仅用于研究展示和风险监测，不构成投资建议。

## 后续提升方向

1. 扩大事件库：增加互动易、招投标、专利、地方政策和长历史新闻。
2. 扩大股票池：纳入更多低空产业链标的并进行分层。
3. 增加第二行情源：补充成交额、换手率、市值、估值和融资融券数据。
4. 增加人工复核：对政策事件和模型高影响事件进行抽样标注。
5. 设置独立最终留出期：验证概率集成和高置信度策略的稳健性。
6. 将系统部署到长期可访问的云端环境。
